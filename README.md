This repository contains the code and aggregated datasets supporting the study
"AI-based early warning of respiratory emergency demand driven by air pollution", submitted to *npj Digital Medicine*.

## Contents
- Python scripts for data preprocessing, statistical analysis, and machine learning
- Aggregated and anonymized datasets
- Reproducible analysis pipeline

## Data availability
Air quality and meteorological data are publicly available from the Chilean National Air Quality Information System (SINCA).
Emergency healthcare data are aggregated and anonymized. All data were extracted from Chilean Department of Health Statistics and Information (DEIS; https://deis.minsal.cl/), Ministry of Health.

## Code availability
All analysis scripts were developed in Python and are provided in this repository.

## Requirements
Python 3.X
See `requirements.txt`.

## Disclaimer
This repository does not contain identifiable patient data.
